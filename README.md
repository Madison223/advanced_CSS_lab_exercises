# advanced_CSS_lab_exercises
This repository contains 19 advanced CSS exercises designed to strengthen web development skill with understanding code and educational learning from a course. 


# Key Topics covered:
- Concepts on Inline and Block displays
- CSS positioning and differences between `static`, `relative`, `absolute`, and `fixed`
- The concepts of floats and clearing elements
- The concepts of padding, margin, and box model
- Fonts, font families, and typography principles
- Principles of visual design
- CSS selectors and targeting specific elements
- The concept of media queries and responsive design for different screens
- Image transformations
- The aspect on mouse hovering effects, transitions, and buttons
- Image filtering and filter styles
- Study of CSS Animation, keyframes, and transformation
- The aspects of CSS frameworks including grid systems, Bootstrap, and the 12-column grid prinicples


# Purpose
These exercises were completed as part of an advanced CSS lab to practice practical styling techniques and reinforce the best practices for modern web development
